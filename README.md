# TaagBack

**TaagBack** is a QR-code-powered scavenger hunt platform. Hunt creators build experiences with clue stops hidden behind printable QR codes; players scan them with the mobile app to progress through the hunt.

## Repository Layout

| Folder | Contents |
|--------|----------|
| `server/` | .NET 10 ASP.NET Core Web API |
| `ui/TaagBack/` | React Native (Expo) mobile app |
| `docs/` | Project documentation |
